## GET /v1/harnesses/{id}

Retrieve a specific harness by its UUID.

### Path Parameters

<ParamField path="id" type="string" required>
  UUID of the harness to retrieve
</ParamField>

### Response

Returns the harness object with all configuration details.

### Example Request

```bash
curl http://localhost:9300/v1/harnesses/01933b5a-0000-7000-8000-000000000001
```

### Example Response

```json
{
  "id": "01933b5a-0000-7000-8000-000000000001",
  "name": "Base Harness",
  "type": "base",
  "description": "Default base harness",
  "capabilities": [
    "session_file_system",
    "virtual_bash"
  ],
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-01-15T10:30:00Z"
}
```

<Note>
  See the [Harnesses concept guide](/concepts/harnesses) for more information about harness types and configuration.
</Note>
